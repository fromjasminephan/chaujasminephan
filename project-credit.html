Business problem

Banks struggle to evaluate loan applicants with no prior credit history — standard scoring models fail entirely when there's no credit track record. This project asked: can behavioral and demographic features alone predict default risk? And how can customers be segmented to enable tailored, data-driven lending strategies?

What we built

  • K-means clustering (k=3) to segment customers into distinct borrower profiles
  • PCA for dimensionality reduction — 90% variance retained
  • Engineered a 3-tier risk label (Low/Medium/High) based on financial indicators
  • SVM classifier with linear kernel — 79% accuracy vs 49% with RBF kernel
  • Bivariate analysis across purpose, housing, job type, age, and duration

Customer segments

  • Cluster 0 — Asset-Heavy Mature Borrowers: Avg age 42, avg loan 6,342 DM, 31+ month terms. Best candidates for secured lending and auto loans.
  • Cluster 1 — Young Independent Starters: Avg age 32, smaller loans ~2,584 DM, shorter terms. Ideal for first-time borrower programs.
  • Cluster 2 — Balanced Mid-Age Homeowners: Avg age 36, 88% own homes. Suited for cross-selling and bundled financial products.

Key findings

  • Savings and checking account status are strongest negative correlators with default risk
  • Job type shows virtually no correlation once other features controlled
  • Silhouette score 0.212, ARI 0.812 — clusters are stable and well-separated
  • All VIF values below 5 — no multicollinearity in the feature set
  • Switching from RBF to linear kernel: +30 percentage point accuracy improvement
